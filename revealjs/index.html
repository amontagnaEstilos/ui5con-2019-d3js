Remember, Remember!

  1. Wrap d3.js visualization as custom control!
  2. Use sap.ui.core.HTML to wrap the SVG element!
  3. Call d3.js from onAfterRendering!
  4. Make it response with sap.ui.core.ResizeHandler!
  5. Use a dummy aggregation to bind against a data model!
